# Cinema Booking Website 🎬🍿

A responsive cinema booking platform inspired by Vox Cinemas. This project features a custom color theme and a fully responsive layout using CSS Media Queries, ensuring a great user experience on both desktop and mobile devices.

## Features
- Dynamic movie browsing
- Responsive layout across all screen sizes
- Custom UI/UX design 

## Tech Stack
- HTML5
- CSS3 (Media Queries for responsiveness)
- JavaScript
- React
